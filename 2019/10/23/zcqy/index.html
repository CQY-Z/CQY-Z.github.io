Git操作指令：

clean
-从对应的远程仓库中克隆内容:__$ git clone [url]__  
-查看当前本地仓库中的文件状态:__$git status__  
-查看当前远程仓库地址:__$ git remote -v__  
-添加远程仓库,与远程仓库连接:__$ git remote add [name] [url]__  
-将本地仓库内容推送到远程仓库:__$ git push [remoteName] [localBranchName]__[^RUNOOB1]  
-添加说明:__$git commit -m "*Explanation*"_  
-拉取远程仓库信息:__$ git pull [remoteName] [localBranchName]__  
-解除与远程仓库的连接:__$ git remote rm [name]__  
-暂存改变或新建的内容:__$ git add [name]__[^RUNOOB2]  
-创键分支:__$git branch [name]__[^RUNOOB3]  
-查看分支:__$git branch -a__  
-切换分支:__$git checkout [name]__  
-合并分支:__$git merge [name]__  
-查看版本号:__$git log__  
-退回到目标版本号:__$git reset --hard 目标版本号  
-显示当前的GIT配置: __$git config --list__ [^RUNOOB4]  
-设置用户名:__$git config --global user.name[username]__  
-设置邮箱地址:__$git config --global user.email[email]__
